# Blog App

> This project is a Blog application built with Ruby and Ruby on Rails

## Built With

- Ruby
- Ruby on Rails
- CSS

## Getting Started

```
git clone https://github.com/wuyepabdul/blog-rails

cd blog-rails

rails db:create
rails db:migrate
rails db:seed

rails server
```

## Tests

```
git clone https://github.com/wuyepabdul/blog-rails
cd blog-rails
bundle install
gem install rspec
rspec
```
